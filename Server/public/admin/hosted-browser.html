X Browser

Standalone Chromium service for X automation. Log in through the password-gated viewer; OreWire posts via API token.

Manual login (password site)

Run the browser on its own process: cd Server && npm run x-browser
Set X_BROWSER_PASSWORD in .env, then open the viewer and sign into X on the live screen.

Open password viewer →

Wire OreWire → X Browser

In Server/.env on the main app:
X_BROWSER_URL=http://127.0.0.1:10088
X_BROWSER_TOKEN= (from x-browser startup log or ~/.orewire-x-browser/api-token.txt)
HOSTED_BROWSER_POST=1
