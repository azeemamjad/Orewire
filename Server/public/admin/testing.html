Backfill filing types

Older filings have no filing_type yet, so per-type counts/selection show 0. This classifies untyped filings from their PDF and fills it in (heuristic, free). Safe to stop and resume — it only fills blanks. Runs in the background; you can leave this page and come back.

Select filings

Prompt & model

Adjust the system prompt and choose the model. DeepSeek models are listed; the active provider default is selected.

Save draft keeps your edits for testing only — it does not change the live system. Save for production replaces the LIVE prompt used to analyze real filings (asks for confirmation first). “Reset to default” loads the built-in prompt into the editor.
